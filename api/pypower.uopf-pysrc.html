 38      """Solves combined unit decommitment / optimal power flow. 
 39   
 40      Solves a combined unit decommitment and optimal power flow for a single 
 41      time period. Uses an algorithm similar to dynamic programming. It proceeds 
 42      through a sequence of stages, where stage C{N} has C{N} generators shut 
 43      down, starting with C{N=0}. In each stage, it forms a list of candidates 
 44      (gens at their C{Pmin} limits) and computes the cost with each one of them 
 45      shut down. It selects the least cost case as the starting point for the 
 46      next stage, continuing until there are no more candidates to be shut down 
 47      or no more improvement can be gained by shutting something down. 
 48      If C{verbose} in ppopt (see L{ppoption} is C{true}, it prints progress 
 49      info, if it is > 1 it prints the output of each individual opf. 
 50   
 51      @see: L{opf}, L{runuopf} 
 52      """ 
 53      ##----- initialization ----- 
 54      t0 = time()                                 ## start timer 
 55   
 56      ## process input arguments 
 57      ppc, ppopt = opf_args2(*args) 
 58   
 59      ## options 
 60      verbose = ppopt["VERBOSE"] 
 61      if verbose:      ## turn down verbosity one level for calls to opf 
 62          ppopt = ppoption(ppopt, VERBOSE=verbose - 1) 
 63   
 64      ##-----  do combined unit commitment/optimal power flow  ----- 
 65   
 66      ## check for sum(Pmin) > total load, decommit as necessary 
 67      on   = find( (ppc["gen"][:, GEN_STATUS] > 0) & ~isload(ppc["gen"]) )   ## gens in service 
 68      onld = find( (ppc["gen"][:, GEN_STATUS] > 0) &  isload(ppc["gen"]) )   ## disp loads in serv 
 69      load_capacity = sum(ppc["bus"][:, PD]) - sum(ppc["gen"][onld, PMIN])   ## total load capacity 
 70      Pmin = ppc["gen"][on, PMIN] 
 71      while sum(Pmin) > load_capacity: 
 72          ## shut down most expensive unit 
 73          avgPmincost = totcost(ppc["gencost"][on, :], Pmin) / Pmin 
 74          _, i = fairmax(avgPmincost)   ## pick one with max avg cost at Pmin 
 75          i = on[i]                     ## convert to generator index

 85          Pmin = ppc["gen"][on, PMIN]
